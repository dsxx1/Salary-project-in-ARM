### 1.2. Соглашения о нотации

Числовые диапазоны записаны как полуоткрытый интервал **[нижняя; верхняя)** — нижняя граница включается, верхняя нет. Это устраняет двусмысленность исходного текста Приказа, где диапазоны заданы по-разному («88–93 %», «до 80 %», «до 49,9 %»).

> ❗ **Округление в Приказе не регламентировано.** Требует решения бухгалтерии до старта (предложение: до копейки, математическое округление). От этого зависит тест «расхождение 0 %».

### 1.3. Сроки

Разработка и внедрение Системы — **не позднее 01.11.2026**.

### 1.4. Цель

Автоматизация расчёта ЗП ~200 сотрудников ОП: замена ручного расчёта (3–5 дней) на формирование ведомости за 2 часа. Вся работа — через интерфейс, без кода со стороны финансиста/ФЭО.

## 2. ТРЕБОВАНИЯ К ВЫПОЛНЕНИЮ РАБОТ

### 2.1. Устраняемые проблемы

| № | Проблема | Решение |
|---|---|---|
| №1 | Смена привязана к юрлицу | Привязка смены к должности |
| №2 | Нет табеля у автоломбарда | Единый табель для всех ОП |
| №3 | Нет контроля замен | Документ закрепления за точкой |
| №4 | Можно пересчитать закрытый период | Статусы периода + запрет изменений |
| №5 | Нет плана → нет расчёта | Правило «нет плана = 100 %» при наличии факта |
| №6 | Баллы теряются при смене юрлица | Баллы привязаны к связке «сотрудник + должность» |
| №7 | Нет истории изменений | Журнал действий + версионирование расчётов |
| №8 | Оклад не привязан к точке | Формула оклада по Приказу СКС-03 |

### 2.2. Описание Работ

#### 2.2.1. Архитектура

```
АРМ → Конструктор витрины (агрегация) → Rule Engine (математика Приказа) → Веб-интерфейс
```
Все параметры (ставки, шкалы, коэффициенты) хранятся в БД и редактируются через экран «Настройки» без изменения кода.

#### 2.2.2. Экран 1 — Табель

Смена содержит: сотрудник, должность, дата, ОП, ТО, юрлицо (только для документов), часы, тип (**обычная/замена**).

**Норма — 17 смен/мес.** С 18-й смены — доплата за сверхнорму (см. 2.2.6.3).

**Лимиты мобильного товароведа** (разд. 9): максимум **3 смены на одном ОП** и максимум **17 смен** за месяц. При превышении — блокировка с показом остатка.

#### 2.2.3. Экран 2 — Закрепление за точкой

Супервайзер закрепляет своих сотрудников за основной точкой.

| Ситуация при открытии смены | Действие |
|---|---|
| Точка в списке закрепления | Обычная смена |
| Точка вне списка → «Это замена?» → Да | Флаг «замена», доплата +42 ₽/ч |
| → Нет | Смена не открывается |

#### 2.2.4. Экран 3 — Планы ТО

Планы вводятся в разрезе ТО и месяца. Карта «показатель → компонент расчёта» (п. 2.1):

| Показатель плана | Куда идёт |
|---|---|
| **Актив** | → Па (основная премия) |
| **Вал** | → Пв (основная премия) |
| **Скупка золота** | → ПсЗ |
| **Актив техники** | → надбавка к Кн / премия КМТ |

Правило «нет плана = 100 %» — см. 2.2.6.1.

#### 2.2.5. Экран 4 — Баллы и коэффициент рейтинга

**Начальные баллы:** при трудоустройстве единовременно начисляется **25 баллов** (п. 1.5.1).
**Максимум:** товаровед-кассир — **70**, продавец-скупщик КМ — **50**. Сверх максимума не начисляется. Минимум — 0.
**Крт берётся по баллам на конец отчётного периода** (п. 1.5, 4.5).
Для баллов с параметром «общий план» сотрудник должен отработать в закреплении ОП **не менее 7 дней**.

#### 2.2.6. Экран 5 — Расчёт ЗП

##### 2.2.6.1. Правило «Нет плана = 100 %»

| Ситуация | Действие |
|---|---|
| Есть факт, нет плана | Выполнение = 100 %, пометка «план не задан, применено 100 %» |
| Есть факт, план = 0 | Выполнение = 0 % |
| Нет факта, нет плана | Ошибка, расчёт заблокирован |

##### 2.2.6.2. Окладная часть (разд. 1)

```
Оклад = (Одн + Одоп) × КЧ × Крт + Нвл
```
- **Одн** — основная часовая ставка; **Одоп** — сумма применимых доп. окладов; **КЧ** — отработанные часы; **Крт** — коэффициент рейтинга; **Нвл** — надбавка за выслугу × число смен.

**Таблица 1 — Показатели для расчёта оклада (₽/час)**

| Показатель | ОП до 12 ч/день | ОП более 12 ч/день |
|---|---|---|
| Основной оклад (Одн) | **150** | **175** |
| Доп. оклад старшего товароведа¹ | 115 | 115 |
| Доп. оклад мобильного товароведа | 156 | 156 |
| Доп. оклад за выдачу займов под залог авто, (А)Ок | 35 | 35 |
| Доп. оклад на подразделениях с КМЗ/КМТ и/или «Ломбард Витрина» | 25 | 25 |

¹ Доп. оклад старшего товароведа выплачивается в сумме **не более 20 000 ₽/мес** (разд. 3).

**Пример:** стаж 1,5 года (Нвл = 150 ₽/смена), Крт = 1,10, 160 ч за 20 смен на ОП до 12 ч/день, без доп. окладов:
`Оклад = (150 + 0) × 160 × 1,10 + 150 × 20 = 26 400 + 3 000 = 29 400 ₽`

##### 2.2.6.3. Доплаты к окладу (вне формулы оклада, Крт не применяется)

| Доплата | Условие | Размер |
|---|---|---|
| Замена в другой нас. пункт (п. 1.4) | Флаг «замена» | +42 ₽/ч |
| Сверхнорма (п. 1.6) | С 18-й смены за месяц | +42 ₽/ч |

**Депремирование старшего товароведа (Таблица 9, разд. 3.1):** удержание **10 %** доп. выплаты при увольнении товароведа по своей инициативе (отработавшего <6 мес) по причине негативного влияния/бездействия старшего, и по иным показателям.

##### 2.2.6.5. Дополнительная премия

**Автоломбард (Пав, п. 2.4.1):**
- +500 ₽ за каждую анкету (любой статус);
- +1 500 ₽ за **первую** анкету (любой статус);
- по заявке с выдачей: **3 %** от суммы выдачи; **4 %** — если заявка оформлена **впервые**;
- кредитному специалисту: **1,5 %** от суммы выдачи; **+0,5 %** при соблюдении/превышении уровня ПСК.

**Скупка техники (Т)ПсТ (п. 2.4.2):** `(Т)Сх × 5 %` (СБП) или `× 3 %` (наличные).

**(А)Кн (по активу авто):** до 700 тыс. → 1 · [700 тыс.; 1,2 млн) → 1,5 · ≥ 1,2 млн → 2.

##### 2.2.6.9. Особые случаи

**Мобильный товаровед (разд. 9):** `Посн = 10 % × П + ПсЗ` (П — премия ОП за смену). Кн не применяется. Лимиты: 3 смены/ОП, 17 смен/мес.

**Подменный товаровед (отпуск по уходу за ребёнком, п. 1.3):** ≥ 8 смен → оклад по Таблице 1; < 8 смен → **175 ₽/ч** фиксированно.

##### 2.2.6.10. Доплаты за привлечение, обучение, наставничество (разд. 5)

- **Обучение стажёра:** 400 ₽/день обучения; +3 000 ₽, если стажёр сдал экзамены и трудоустроился (делится между обучавшими пропорционально дням).
- **Наставничество:** +3 000 ₽ после испытательного срока курируемого; ещё +3 000 ₽ после 4 мес работы.
- **Привлечение:** 3 000 ₽ за трудоустройство рекомендованного; +3 000 ₽ после прохождения им испытательного срока; для КМЗ — при отработке ≥ 2 мес и плане ≥ 90 %.

##### 2.2.6.11. МГВ — выполняется последней (разд. 10)

Условие: отработано **≥ 14 календарных дней**. Иначе берётся результат предыдущих шагов.
```
Итоговая ЗП = MAX(результат расчёта, МГВ)
```

| Тип | Ставка | Максимум |
|---|---|---|
| Обычный | 310 ₽/ч × часы | 55 800 ₽/мес |
| Мобильный товаровед | 390 ₽/ч × часы | 70 200 ₽/мес |

К МГВ не применяются Нвл и Крт. Действует с 01.09.2025.

1. **Округление** — в Приказе не задано. Решение бухгалтерии (предложение: до копейки, мат. округление). Критично для теста «0 %».
